# Government Macro Assumption Tables: Where To Pull Them

## 1) Parliament project pages (primary source for budget-framework assumptions)

For the **2026 Budget Framework Statement / 2027-2028 projections**, use the Parliament project page and its attached files:

- Project page: `https://sforum.parliament.mn/projects/28295`
- Law project attachment: `https://sforum.parliament.mn/files/44b8f7b8-b1c6-432f-8cb7-87d46a4846f8`
- Explanatory attachment: `https://sforum.parliament.mn/files/62a74e1e-bf6f-4818-a4f4-0b7ccf66bb89`

The law-project attachment includes the headline macro block (e.g., GDP growth and inflation path).
The explanatory attachment discusses the modeling assumptions, including exchange-rate/inflation/export-import assumptions.

## 2) Fiscal Stability Council macro report submitted to Parliament

The Parliament news page for 2026-2028 macro forecast package:

- `https://www.parliament.mn/nn/70154/`
- Attachment URL observed from that page: `https://www.parliament.mn/files/5a772a2a054f45d3bc5988e4a82accfa/?d=1`

This report is useful for scenario logic and external assumptions (commodity prices, external sector, exchange-rate path in model outputs), but it is not a direct substitute for formal budget-law annex tables.

## Practical note

- Some attachments are image-heavy/scanned PDFs. Plain text extraction can miss table cells.
- For publishable numbers, keep the original file and manually verify key cells (USD/MNT, inflation, exports/imports) against the table image.
